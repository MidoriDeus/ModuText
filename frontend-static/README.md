# ModuText - Frontend Estático

Versión estática del frontend de la aplicación ModuText para demostración en Vercel.

## Acerca de esta versión

Esta es una versión estática del frontend de ModuText que contiene todas las páginas principales con su diseño y funcionalidad visual intactos, pero sin la parte de backend en PHP. Esta versión permite:

- Visualizar el diseño completo de la aplicación
- Demostrar la navegación entre páginas
- Mostrar la interfaz de usuario con datos simulados
- Funcionalidad básica con SweetAlerts simulados

## Páginas incluidas

- `index.html` - Página principal con presentación, historia, misión, visión y objetivos
- `login.html` - Página de inicio de sesión y registro de usuarios
- `retiro.html` - Formulario para solicitudes de retiro de telas
- `sucursales.html` - Gestión de direcciones de retiro

## Características

- Diseño completamente responsivo
- Interacciones con SweetAlert2
- Animaciones y transiciones CSS
- Formularios funcionales con validación simulada
- Contadores animados
- Carrusel de imágenes

## Limitaciones

Esta versión es completamente estática, por lo tanto:

- No hay persistencia de datos real
- Las operaciones de backend están simuladas
- No hay conexión a base de datos
- Las funcionalidades críticas están emuladas

## Uso en Vercel

Para desplegar en Vercel:

1. Sube esta carpeta completa a tu repositorio
2. Conecta el repositorio con Vercel
3. Asegúrate de que el directorio raíz tenga el archivo `vercel.json`
4. El despliegue se hará automáticamente
